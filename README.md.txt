Hola!

El presente script pretende ser una pequeña contribución a la comunidad para aquellos que estén buscando realizar un análisis de tipo NLP con Python.
El objetivo consistió en analizar la opinión pública que ronda la palabra "Argentina" en Twitter.
Para ello, se accedió a las publicaciones en las cuales se mencionaba dicha palabra, utilizando snscrape, en una ventana de tiempo determinada, 
extrayendo la información que conformó el dataset.

En progreso.
